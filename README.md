# 🗺️ Korattur Temples GIS Dataset  
### 📍 Spatial Analysis using QGIS

---

## 📖 Overview
This project presents a **GIS-based spatial dataset** of temples located in **Korattur, Chennai**.  
It focuses on mapping and analyzing the distribution of:

- 🛕 Shiva Temples  
- 🛕 Murugan Temples  
- 🛕 Vinayagar Temples  

---

## 📊 Dataset Information

| Field | Description |
|------|------------|
| id | Unique ID |
| name | Temple Name |
| type | Shiva / Murugan / Vinayagar |
| latitude | Geographic coordinate |
| longitude | Geographic coordinate |

---

## 🗺️ How to Use (QGIS)

1. Open QGIS  
2. Go to **Layer → Add Layer → Add Delimited Text Layer**  
3. Select dataset  
4. Set:
   - X → longitude  
   - Y → latitude  
   - CRS → EPSG:4326  
5. Click **Add**
